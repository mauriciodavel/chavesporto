# 🚀 Início Rápido - Painel de Ambientes

## ⚡ 5 Minutos para Começar

### 1️⃣ Instalar Multer
```bash
cd backend
npm install
```

### 2️⃣ Iniciar Servidor
```bash
npm run dev
```

### 3️⃣ Acessar Painel
```
http://localhost:3000/painel
```

✅ **Pronto!** O painel já está funcionando.

## 📞 Problemas Comuns

| Problema | Solução |
|----------|---------|
| Painel vazio | Crie reserva aprovada para hoje |
| Multer error | `npm install multer` |
| CSS ruim | Limpe cache (Ctrl+Shift+Del) |
| Admin login falha | Confirme role = 'admin' |
